This server is a Raspberry Pi 3B+. It runs Ubuntu Server, Apache 2, SFTPGo, OpenLDAP, LdapCherry, Postfix, Dovecot and Roundcube. The procedure for configuring it is described here.

The server is intended for use in an AREDN mesh. It enables users to exchange email with each other, and with people in the public Internet. Users can access their email via a web-based UI, IMAP or POP. People can create their own user accounts via a web-based UI.

The server also provides chat and file sharing.
