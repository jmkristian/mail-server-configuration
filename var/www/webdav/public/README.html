This is a place to share files. Anyone can read any file.

You can browse files starting at the URL path http:/files/. You can upload and download files via FTP (port 21), WebDAV on HTTP port 2121 or the web UI at http:/edit-files . You can log in with the username "anonymous" and any password. If you log in with any other username, you can modify files in the folder whose name is your username. To get a username, you can create an account.

You can access WebDAV from Windows File Explorer by applying "Map Network Drive" to a folder name like http://hostname:2121 . But this won't work until you use regedit to set this registry key = 2: 

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WebClient\Parameters\BasicAuthLevel
The value 2 enables Windows to send a cleartext password via HTTP. This WebDAV server requires a cleartext password, because the FCC forbids encryption over amateur radio.
