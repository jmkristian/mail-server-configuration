This is a place to share files. Anyone can read any file.

You can browse files starting at the URL path http:/files/. These URLs are useful for sharing files with other people. For example, you can paste a URL into an email message, and the message recipients can click the link to browse or download the file or folder.

You can access files via FTP (port 21), WebDAV on HTTP port 2121 or the web-based UI at http:/edit-files . You can log in with the username "anonymous" and any password. If you log in with any other username, you can upload files into the folder whose name is your username. To get a username, you can create an account.

You can access files in Windows File Explorer by applying "Map Network Drive" to a folder name like http://hostname:2121 (the WebDAV server). But this won't work until you set this registry key = 2 (using regedit): 

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WebClient\Parameters\BasicAuthLevel
The value 2 enables Windows to send a cleartext password via HTTP. This WebDAV server requires a cleartext password, because the FCC forbids sending encrypted data over amateur radio.
